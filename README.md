Objective-C Koans
=================

Originally created by [Joe Cannatti] [1], inspired by the reknowned [Ruby Koans] [2].

Running the Koans
-----------------

# git clone this repo
# Open the XCode project
# Run the tests with menu: Product->Test or Command-U
# Make each test pass to learn about Objective-C
